# Project Report

## Objective

Build an automated CI/CD pipeline using Jenkins.

## Tools Used

- Jenkins
- SonarQube
- Nexus
- Docker
- Kubernetes

## Workflow

1. Developer pushes code to GitHub
2. Jenkins triggers pipeline
3. SonarQube scans code quality
4. Nexus stores artifacts
5. Docker builds container image
6. Kubernetes deploys application

## Lessons Learned

- Pipeline automation
- Docker containerization
- Kubernetes deployment
